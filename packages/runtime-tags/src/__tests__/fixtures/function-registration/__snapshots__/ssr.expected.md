# Write
```html
  <div>3</div><div>4</div>
```

# Render End
```html
<html>
  <head />
  <body>
    <div>
      3
    </div>
    <div>
      4
    </div>
  </body>
</html>
```

# Mutations
```
INSERT html
INSERT html/head
INSERT html/body
INSERT html/body/div0
INSERT html/body/div0/#text
INSERT html/body/div1
INSERT html/body/div1/#text
```
