# Write
```html
  <header class=my-header>Header content</header><main>Body content</main><footer class=my-footer>Footer content</footer>
```

# Render End
```html
<html>
  <head />
  <body>
    <header
      class="my-header"
    >
      Header content
    </header>
    <main>
      Body content
    </main>
    <footer
      class="my-footer"
    >
      Footer content
    </footer>
  </body>
</html>
```

# Mutations
```
INSERT html
INSERT html/head
INSERT html/body
INSERT html/body/header
INSERT html/body/header/#text
INSERT html/body/main
INSERT html/body/main/#text
INSERT html/body/footer
INSERT html/body/footer/#text
```
