# Write
```html
  <div><span>1</span></div>
```

# Render End
```html
<html>
  <head />
  <body>
    <div>
      <span>
        1
      </span>
    </div>
  </body>
</html>
```

# Mutations
```
INSERT html
INSERT html/head
INSERT html/body
INSERT html/body/div
INSERT html/body/div/span
INSERT html/body/div/span/#text
```
