# Render
```html
<div>
  <button>
    0
  </button>
</div>
```

# Update
```js
container.querySelector("button").click();
```
```html
<div>
  <button>
    1
  </button>
</div>
```
## Change
```
UPDATE: div > button::text "0" => "1"
```
